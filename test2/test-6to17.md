# Test 8

**Answer** :

MVC Model-View-Controller
เป็น pattern การเขียนที่ได้รับความนิยม
Model กำหนดโครงสร้าง object

Controller เป็นตัวกลาง ใช้กำหนด data flow ทำการ update view
